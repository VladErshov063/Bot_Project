# Telegram-бот для изучения китайского языка 🇨🇳

Бот помогает учить китайские слова по уровням HSK. Он выделяет из любого китайского текста незнакомые слова, создаёт очередь для изучения и использует систему интервальных повторений для долговременного запоминания. Также доступно **5 обучающих игр** для закрепления материала.

## 🚀 Возможности

- **Определение уровня HSK** – вы задаёте свой уровень, бот считает слова выше этого уровня новыми.
- **Автоматическое выделение слов** – при отправке китайского текста бот токенизирует его, сверяет с эталонными словарями HSK и добавляет неизвестные слова в очередь.
- **Карточки для изучения** – каждое новое слово показывается с иероглифом, пиньинем, переводом, уровнем HSK и контекстом (предложением из вашего текста).
- **Система интервальных повторений** – изученные слова планируются к повторению по алгоритму, учитывающему сложность слова и коэффициент забывания.
- **5 игр для повторения**:
  - 📝 *Угадай перевод* – выберите правильный перевод из четырёх вариантов.
  - 🔄 *Перемешанные иероглифы* – соберите слово из перемешанных иероглифов.
  - 🔊 *Выбери иероглиф по пиньиню* – найдите правильный иероглиф по произношению.
  - ⛓️ *Цепочка слов* – продолжайте цепочку слов по последнему иероглифу.
  - 🎵 *Выбери правильный пиньинь* – тренируйте тоны, выбирая корректное произношение.
- **Статистика** – вы всегда видите, сколько слов уже выучено и сколько ждёт своей очереди.
- **Словарик** – список всех изученных слов с переводом.
- **Главное меню с кнопками** – удобный доступ ко всем функциям.

## 📋 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие, показ текущего уровня HSK |
| `/level <1-6>` | Установить свой уровень HSK |
| `/add_text` | Отправить китайский текст для анализа (бот переходит в режим ожидания) |
| `/learn` | Показать следующее новое слово из очереди |
| `/review` | Запустить сессию повторения слов |
| `/stats` | Показать статистику (уровень, изучено слов, в очереди) |
| `/mydict` | Показать все изученные слова с переводом |
| `/game` | Открыть меню выбора игр |
| `/reset` | Полностью очистить все изученные слова и очередь (с подтверждением) |
| `/cancel` | Отменить текущий диалог |
| `/help` | Показать это сообщение |

*Все команды также доступны через кнопки под полем ввода.*
