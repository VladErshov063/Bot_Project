# Telegram-бот для изучения китайского языка 🇨🇳

Бот помогает учить китайские слова по уровням HSK. Он выделяет из любого китайского текста незнакомые слова (на основе вашего уровня), создаёт очередь для изучения и использует систему интервальных повторений для долговременного запоминания.

## 🚀 Возможности

- **Определение уровня HSK** – вы задаёте свой уровень (1–6), бот считает слова выше этого уровня «новыми».
- **Автоматическое выделение слов** – при отправке китайского текста бот токенизирует его, сверяет с эталонными словарями HSK и добавляет неизвестные слова в очередь.
- **Карточки для изучения** – каждое новое слово показывается с иероглифом, пиньинем, переводом, уровнем HSK и контекстом (предложением из вашего текста).
- **Система интервальных повторений** – изученные слова планируются к повторению по алгоритму, учитывающему сложность слова и коэффициент забывания.
- **Статистика** – вы всегда видите, сколько слов уже выучено и сколько ждёт своей очереди.

## 📋 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие, показ текущего уровня HSK |
| `/level <1-6>` | Установить свой уровень HSK |
| `/add_text` | Отправить китайский текст для анализа (бот переходит в режим ожидания) |
| `/learn` | Показать следующее новое слово из очереди |
| `/review` | Запустить сессию повторения слов |
| `/stats` | Показать статистику (уровень, изучено слов, в очереди) |
| `/cancel` | Отменить текущий диалог |
